# Customer Insights Dashboard (SQL + Power BI)

## 📊 Project Overview

This project analyzes customer behavior and revenue contribution using SQL and Power BI. It provides insights into top customers, regional distribution, and category performance.

## 🚀 Key Features

* KPI metrics: Total Customers, Total Sales, Avg Revenue per Customer
* Customer trend analysis over time
* Top 10 customers by revenue
* Region-wise customer distribution
* Category-wise sales analysis
* Interactive slicers (Region, Category, Date)

## 🛠️ Tools Used

* SQL Server (data extraction & analysis)
* Power BI (dashboard & visualization)
* DAX (calculated measures)
* Power Query (data transformation)

## 📁 Files Included

* Customer Insights Dashboard (.pbix)
* Dataset (.csv)
* Dashboard Screenshot
